# realworld-api

## 기술 스택
Language : `golang`
Database : `AWS RDS(MariaDB)` `redis`
ORM : `gorm`
Infra : `AWS ECS` `Docker` `Github Actions`
Logging : `aws cloudwatch`
API Documentation : `swagger`
Error Tracking : `sentry`
